# Base64 Encoder / Decoder

## Encoder
1. 문자열 입력
2. 입력받은 문자열을 ASCII코드로 변경
3. ASCII코드(십진수)를 이진수로 변경
4. 이진수를 6자리씩 잘라줌
5. 잘라준 크기가 4의 배수가 되지 않는다면 마지막 이진수의 크기를 6으로 맞춰주고 크기를 4의 배수가 될 때 까지 '='으로 채워줌
6. 이진수를 십진수로 바꾸어 줌
7. 십진수로 base64에 맞는 문자를 결과에 저장
   - 만약 '='라면 그대로 저장

## Decoder
1. 문자열 입력
2. 입력받은 문자열을 base64에서 숫자로 변경
3. 변경한 숫자(십진수)를 이진수로 변경
4. 이진수를 8자리씩 잘라줌
   - 만약 8자리로 자르고 남는 부분이 있다면 무시(0이거나 '='일 경우임)
5. 잘라준 이진수를 십진수로 변경 
6. ASCII코드 상의 문자를 결과에 저장
